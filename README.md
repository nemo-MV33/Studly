# Studly

> Продолжение моего проекта **Noty**: из простого трекера напоминаний в полноценный учебный планер для школьников и студентов.

![Studly Preview](docs/preview.png)

## Что это

**Studly** помогает держать в одном месте домашние задания и напоминания по учебе:
- по дням недели и датам,
- по предметам с цветовой маркировкой,
- с вложениями (фото, аудио, документы).

Приложение сделано под iOS и сфокусировано на быстром ежедневном использовании.

## Основные возможности

- Два основных раздела: `Планер` и `Настройки`.
- Предметы с цветными индикаторами.
- Календарная навигация по неделям + быстрый переход к дате.
- Создание ДЗ и напоминаний с датой/временем и локальными уведомлениями.
- Поиск по всем задачам (название, дата/время, предмет).
- Отметка выполнения без удаления задачи.
- Пометка задачи как `Важное`.
- Swipe actions для задач:
  - влево: удаление,
  - вправо: важное / редактирование.
- Детальная карточка задачи по тапу.
- Вложения:
  - фото (горизонтальная лента, fullscreen, свайп между фото, zoom),
  - аудио (встроенный плеер),
  - документы (`pdf`, `docx`, `pptx` и др.) через встроенный просмотр.
- Хранение вложений в песочнице приложения (`Documents/attachments`).

## Запуск

### В Xcode

1. Открой `Studly.xcodeproj`.
2. Выбери схему `Studly`.
3. Запусти на симуляторе или устройстве.

### Через CLI

```bash
xcodebuild -project Studly.xcodeproj -scheme Studly -sdk iphonesimulator -configuration Debug build CODE_SIGNING_ALLOWED=NO
```
